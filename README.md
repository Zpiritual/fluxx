fluxx
=====


Digital version of the game fluxx


Project Organisation
====================
--src 		(.cc) 


--include	(.h)


--test		(cc unit test)


--doc		(docs)


--bin		(binaries)


--lib		(external dependencies)

HOW TO RUN TESTS
================
```cmd
g++ -std=c++11 -o [testfile] -I include test/[testfile].cc
[testfile].exe
```
